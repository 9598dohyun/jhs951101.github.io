사전 캠페인 참여 기능

크리에이터들이 사전 캠페인에 참여할 수 있습니다.

사회에 기여하려고 한 점
크리에이터들이 캠페인에 참여하면서 콘텐츠 소재를 늘릴 수 있고, 한류를 좋아하는 외국인들한테 크리에이터들을 소개할 수 있습니다.
배경과 목적
'(주)이엔티팩토리'회사에서 프리랜서로 근무하면서 프로젝트에 참여하였습니다.
배운 점
  • IntelliJ에서 Spring 프로젝트를 실행하고, HTTP GET 요청 처리 함수를 추가하는 법을 알게 되었습니다.
  • Mybatis를 이용해서 MySQL과 연동하는 법을 알게 되었습니다.
  • DB Migration으로 테이블을 추가, 수정, 삭제하는 법을 알게 되었습니다.
  • 구글, 페이스북, 틱톡 API를 활용하는 법을 알게 되었습니다.
