Background and Purpose
One of the hottest things these days is IoT. Therefore, I did my own project to develop basic knowledge in preparation for becoming an IoT developer.
What I Learned
  • I learned how to send HTTP Request from RaspberryPi and Electron, and the difference between GET and POST.
  • I learned how to use LED after installing it on RaspberryPi.
  • I learned how to implement a client with Electron
  • I learned how to operate a server after building it on AWS and Cafe24.
  • I learned how to send HTTP Response in PHP according to the situation.
  • I learned how to build a server with PHP and how to build and connect a database with MariaDB.
Process
  1. I implemented the client with Electron. At this time, HTTP Request is sent.
  2. I built a server with PHP, built a database with MariaDB, and connected it.
  3. I started a server on AWS and Cafe24. However, when launching to AWS, an error occurred that even accessing the database was not possible. So, after creating a new MariaDB user, I gave this user full privileges and used it on the server. This allows access to the database.
  4. In this way, I was able to implement the program I wanted.
