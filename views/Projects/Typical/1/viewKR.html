구현 기능

회사명을 통한 제품 리스트 조회 기능

회사명을 입력하면, 해당 회사의 제품 리스트를 조회할 수 있습니다.

평가 점수 계산 기능

인증 평가 항목을 바탕으로 점수를 모의로 계산할 수 있습니다.

기자재 추천 기능

지금까지 입력한 정보를 바탕으로 기자재를 추천 받을 수 있습니다.

결과 화면
사회에 기여하려고 한 점
녹색 건축 인증 절차를 최대한 간단하게 할 수 있고, 평면도 등의 개인 정보가 유출되는 것을 방지할 수 있습니다.
배경과 목적
'디어그린'회사에서 근무하면서 프로젝트에 참여하였습니다.
배운 점
  • PHP, JavaScript, JQuery, Ajax로 웹페이지에 기능을 구현하는 법을 알게 되었습니다.
  • PHP에서 MySQLi를 이용해서 MySQL과 연동하는 법을 알게 되었습니다.
  • 홈페이지와 웹서버를 Cafe24에 띄운 후 연동하는 법을 알게 되었습니다.
  • Python에서 symbols, diff로 미적분 공식을 만든 후 결과를 출력하는 법을 알게 되었습니다.
  • Flask로 구축한 서버를 AWS에 띄운 후 홈페이지와 연동하는 법을 알게 되었습니다.
  • 한국환경산업기술원, 한국에너지공단, 도로명주소 API를 활용하는 법을 알게 되었습니다.
역량 강화를 위해 가장 노력한 점
PHP, JQuery, Ajax로 한국환경산업기술원, 한국에너지공단 API 활용
전체 구조 및 사용 기술

  • PHP: 홈페이지와 서버를 구축하였습니다.
  • MariaDB: 데이터베이스를 구축하고 공공데이터를 활용하였습니다.
  • Flask: 데이터 분석기를 홈페이지와 연동하기 위한 서버를 구축하였습니다.
  • Cafe24: 홈페이지를 띄우는 호스팅 서비스로 사용하였습니다.
  • AWS: 데이터 분석기를 사용할 수 있도록 하는 호스팅 서비스로 사용하였습니다.
개발 환경
  • Visual Studio Code: 웹페이지와 서버를 구현하기 위해서 사용하였습니다.
  • FileZilla: 카페24에 접속하기 위해서 사용하였습니다.
  • TablePlus: DB에 접속하기 위해서 사용하였습니다.
  • Terminal: AWS에 접속하기 위해서 사용하였습니다.
