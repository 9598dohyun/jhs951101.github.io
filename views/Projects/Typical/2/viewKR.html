결과 화면

사회에 기여하려고 한 점
한식을 추천 받으므로 한국을 방문하는 외국인들이 더 즐겁게 관광할 수 있습니다.
배경과 목적
'KNOW'회사에서 근무하면서 프로젝트에 참여하였습니다.
배운 점
  • Axios로 네트워크 통신, Expo-camera로 사진 촬영 등 React Native로 페이지에 기능을 구현하는 법을 알게 되었습니다.
  • PHP에서 MySQLi를 이용해서 MySQL과 연동하는 법을 알게 되었습니다.
  • 서버를 Cafe24에 띄운 후 연동하는 법을 알게 되었습니다.
  • 한국관광공사, 서울관광재단, 카카오 API를 활용하는 법을 알게 되었습니다.
역량 강화를 위해 가장 노력한 점
React Native, PHP로 한국관광공사, 서울관광재단, 카카오 API 활용
전체 구조 및 사용 기술

  • React Native (Android or iOS): 하이브리드 앱을 구현하였습니다.
  • PHP: 서버를 구축하였습니다.
  • MariaDB: 데이터베이스를 구축하고 공공데이터를 활용하였습니다.
  • Cafe24: 서버를 띄우는 호스팅 서비스로 사용하였습니다.
개발 환경
  • Visual Studio Code: 하이브리드 앱과 서버를 구현하기 위해서 사용하였습니다.
  • FileZilla: 카페24에 접속하기 위해서 사용하였습니다.
  • TablePlus: DB에 접속하기 위해서 사용하였습니다.
  • Notion, Google Drive: 팀원들과 주간 회의 자료를 공유하기 위해서 사용하였습니다.
  • Figma: 디자이너와 협업하기 위해서 사용하였습니다.
  • Google Analytics for Firebase: 사용자들의 앱 사용에 관한 통계를 내기 위해서 사용하였습니다.
