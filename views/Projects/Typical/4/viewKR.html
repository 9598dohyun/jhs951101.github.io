배경과 목적
'(주)디어그린'회사에서 근무하면서 프로젝트에 참여하였습니다.
배운 점
  • HttpWebRequest로 네트워크 통신 등 WPF로 페이지에 기능을 구현하는 법을 알게 되었습니다.
  • PHP에서 MySQLi를 이용해서 MySQL과 연동하는 법을 알게 되었습니다.
  • 웹서버를 Cafe24에 띄운 후 연동하는 법을 알게 되었습니다.
  • Flask로 구축한 서버를 AWS에 띄운 후 홈페이지와 연동하는 법을 알게 되었습니다.
전체 구조 및 사용 기술

  • WPF(C#): Windows 앱을 구현하였습니다.
  • PHP: 서버를 구축하였습니다.
  • MariaDB: 데이터베이스를 구축하고 공공데이터를 활용하였습니다.
  • Flask: 데이터 분석기를 홈페이지와 연동하기 위한 서버를 구축하였습니다.
  • Cafe24: 홈페이지를 띄우는 호스팅 서비스로 사용하였습니다.
  • AWS: 데이터 분석기를 사용할 수 있도록 하는 호스팅 서비스로 사용하였습니다.
개발 환경
  • Visual Studio 2017: Windows 앱을 구현하기 위해서 사용하였습니다.
  • Visual Studio Code: 서버를 구현하기 위해서 사용하였습니다.
  • FileZilla: 카페24에 접속하기 위해서 사용하였습니다.
  • TablePlus: DB에 접속하기 위해서 사용하였습니다.
  • Terminal: AWS에 접속하기 위해서 사용하였습니다.
  • Figma: 디자이너와 협업하기 위해서 사용하였습니다.
  • Google Drive: 팀원들과 회의 자료를 공유하기 위해서 사용하였습니다.
